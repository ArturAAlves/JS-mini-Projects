Resets

O CSS Reset é - basicamente - um arquivo que “limpa” os estilos padrões de todos os elementos HTML que já possuem uma formatação padrão. Ele tira cores, tamanhos de fonte, margins, paddins, efeitos e decorações… Quase tudo de quase todos os elementos vira “padrão”, como um texto sem formatação/alteração nenhuma.


*, *::before,*::after {
    box-sizing: border-box;
}

html {
    font-family: sans-serif;
     line-height: 1.15;
    -webkit-text-size-adjust: 100%;
}

 article, aside, figcaption, figure, footer, header, hgroup, main, nav, section {
    display: block;
}

ul {
    list-style: none;
    padding: 0;
    margin: 0;
} 

a {
    text-decoration: none;
}

Alinhamentos

Neste ponto, vamos focar os pontos mais comuns referentes ao alinhamento de elementos em HTML

text-align: Alinha o conteudo contido em um elemento do tipo "block".

  • center
  • right
  • center
  • justify
  • inherit

Temos também o propriedade "direction" que dá a direção a seguir

  • direction:ltr;
  • direction:rtl;

Alinhamento de imagens

Um dos shorthands comuns 


display: block;
margin:auto;

ou quando inserida em um <p>:


text-align: center;

O seguinte exemplo serve para alinhar uma DIV ao centro do ecran


#section {
    position: relative;
    height: 100vh;
    width: 100vw;
    text-align: center;
}

.#section-center{
    position: absolute;
    top: 50%;
    left: 50%;
    transform: translate(-50%,-50%);
}

Uma outra forma, pode ser utilizando flexbox, segue o exemplo


display: flex;
align-items: center;
justify-content: center }
Posicao elements

A utilização de Position, server para especificar o metodo de posicionamento usado para cada elemento

  • Static - valor pré-difinido
  • absolute - o elemento é posicionado relativamente ao seu parente superior
  • fixed - o elemento é posicionado relativamente á janela do browser
  • relative - o elemento é posicionado relativamente a sua posiçao normal
  • sticky - o elemento é posicionado relativamente a posiçao do scroll do utilizador.
    É uma mistura entre posição "relatiive" e "fixed"
  • initial - o elemento é posicionado na sua posição default
  • inherit - o elemento herda as propriades do parente superior
